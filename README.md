# Expense Tracker DevOps Project

## Project Overview

A complete end-to-end DevOps implementation of an Expense Tracker application using FastAPI, Docker, Terraform, Azure, Ansible, Prometheus, and Grafana.

## Tech Stack

- FastAPI
- Python
- SQLite
- Docker
- Docker Compose
- GitHub
- GitHub Actions
- Terraform
- Azure VM
- Ansible
- Prometheus
- Grafana
- Node Exporter
